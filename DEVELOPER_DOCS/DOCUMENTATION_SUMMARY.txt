ðŸ’¡ KEY CONCEPTS TO UNDERSTAND:
================================================================================

1. Sphinx Domain
   - This project adds a "domain" to Sphinx for documenting MATLAB
   - Similar to how Sphinx documents Python, C++, etc.

2. Documenters
   - Extract documentation from MATLAB source files
   - Generate reStructuredText content for Sphinx

3. Type System (mat_types.py)
   - Represents MATLAB code elements (classes, functions, etc)
   - MatClass, MatFunction, MatProperty, etc.

4. Parser (mat_tree_sitter_parser.py)
   - Uses tree-sitter to parse MATLAB syntax
   - Converts AST to our type objects

5. Testing
   - Uses pytest with Sphinx test fixtures
   - Tests are in tests/ directory
   - Run with: pytest
